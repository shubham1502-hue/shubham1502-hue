# LinkedIn Profile Playbook

Goal: make the profile read like a founder-facing operator with proof of work, not a job seeker with projects.

## Current Diagnosis

The current profile is directionally strong:

- clear Founder Office / RevOps direction
- strong STEMpedia proof point
- GitHub proof-of-work included
- About section has a distinctive point of view

The weak spots:

- headline spends too much space saying "looking for roles"
- About section is strong but dense
- Featured section should lead with the founder operating-system toolkit, not market sizing
- Experience section hides the public Founder OS work unless someone reads the About section
- Skills should emphasize operator systems, not only tools

## Recommended Headline

Use one of these.

Best:

```text
Building founder-facing operating systems for early-stage startups | Founder's Office, RevOps & GTM Systems | Ex-STEMpedia
```

More direct:

```text
Founder's Office & RevOps Operator | Building reusable systems for GTM, metrics, investor updates and startup execution
```

If you want to keep the job-search signal:

```text
Founder's Office & RevOps Operator | Building founder-facing operating systems | Open to early-stage startup roles
```

Avoid:

```text
Looking for Founder's Office/adjacent roles
```

It makes the profile feel applicant-led. Lead with the value, not the ask.
